# 启动 Minecraft

## 选择游戏版本

1. 在启动器启动页的版本列表中选择想要启动的版本
2. 点击版本名称选中该版本

## 启动游戏

1. 确保您已经添加了 Minecraft 角色
2. 确保您已经选择了正确的游戏版本
3. 点击 "启动游戏" 按钮
4. 等待游戏加载完成
5. 进入 Minecraft 主界面

## 首次进入游戏

1. 选择 "单人游戏" 或 "多人游戏"
2. 如果想单人游戏，可以：
   - 创建新世界
3. 如果想多人游戏，可以：
   - 添加服务器

## 常见问题

### 游戏无法启动

- 检查您是否安装了正确版本的 Java (低版本建议使用 Java 8, 高版本建议使用 Java 21+)
- 检查游戏内存分配是否合理
- 检查是否安装了必装的驱动程序
- 查看启动器日志获取详细错误信息,进入反馈群:1002593659后 @咸鱼

### 游戏崩溃

- 检查是否安装了不兼容的模组
- 检查游戏版本和加载器版本是否匹配
- 查看崩溃报告获取详细信息
