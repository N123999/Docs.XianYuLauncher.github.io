# Mod管理

## 安装加载器

要使用Mod，您需要先安装加载器。请参考 [下载和安装 Minecraft](./download-minecraft.md) 中的 "安装加载器" 部分。

## 浏览和下载Mod

### 通过Mod下载

1. 在启动器资源下载页，点击顶部导航栏的 "Mod" 按钮
2. 在Mod页中浏览可用的Mod
3. 可以通过以下方式筛选Mod：
   - 类型
   - 加载器
   - 版本
4. 点击Mod卡片查看详情
5. 选择要安装的版本，点击 "下载" 按钮
6. 选择要安装到的游戏版本
7. 等待下载和安装完成(如您开启了 "自动下载前置" 功能, 则会自动下载前置Mod)

### 手动安装Mod

1. 从第三方网站下载Mod文件（.jar 格式）
2. 在启动器版本管理页，将Mod文件拖放至启动器窗口并松开
7. Mod会自动显示在Mod列表中

## 管理已安装的Mod

1. 在启动器版本列表页，选择安装了Mod的游戏版本
3. 在版本管理中，找到 "Mod" 按钮
4. 可以：
   - 启用/禁用Mod
   - 查看Mod详情
   - 更新Mod
   - 转移Mod
   - 删除Mod
