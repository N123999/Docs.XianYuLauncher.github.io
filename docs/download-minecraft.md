# 下载和安装 Minecraft

## 下载 Minecraft 版本

1. 在启动器侧边栏，点击 "下载" 图标
2. 在资源下载页中选择您想要下载的版本
3. 可以选择：
   - 最新正式版
   - 最新快照版
   - 其它版本
4. 选择版本后，点击下载Minecraft页的 "下载" 按钮
5. 等待下载完成

## 安装加载器（可选）

如果您想要安装模组，可以选择安装加载器：

1. 在下载版本时，选择加载器
2. 在加载器选项中，选择您想要安装的加载器：
   - Forge
   - Fabric
   - Neoforge
   - Quilt
3. 选择加载器版本
4. 点击 "下载" 按钮
5. 等待下载和安装完成

## 版本安装完成后

- 安装完成的版本会出现在启动页的版本列表中
- 可以选择该版本进行启动
