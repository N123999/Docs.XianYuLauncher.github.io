# FAQ

## Launcher Issues

### Launcher Fails to Start

**Problem**: The launcher doesn't respond when double-clicking the shortcut.

**Solutions**:
- Check if your system meets the minimum requirements (Windows 10/11 X64)
- Check if .NET 7.0 Runtime and WinAppSDK 1.4 are installed
- Check the permissions of the launcher installation directory
- Reinstall the launcher or join our QQ group: 1002593659 and contact @咸鱼 for feedback

### Launcher Crashes

**Problem**: The launcher crashes during operation.

**Solutions**:
- Check if the launcher is up to date
- Close other memory-intensive programs
- Check the launcher logs for detailed error information

## Minecraft Issues

### Game Fails to Launch

**Problem**: The game doesn't respond or closes immediately after clicking the "Launch Game" button.

**Solutions**:
- Check if Java is installed correctly
- Adjust game memory allocation
- Check if incompatible mods are installed
- Check game logs for detailed error information

### Game Lags

**Problem**: The game experiences lag or frame drops during runtime.

**Solutions**:
- Lower game graphics settings
- Increase game memory allocation
- Close background programs
- Delete some unused mods
- Optimize game settings

### Game Crashes

**Problem**: The game crashes during runtime.

**Solutions**:
- Check if incompatible mods are installed
- Check if the game version matches the loader version
- Try updating the game to the latest version
- Try updating the loader to the latest version
- Check the crash report for detailed information

## Account Issues

### Cannot Log In

**Problem**: Cannot log in to the Minecraft account after entering account information.

**Solutions**:
- Check if the account password is correct
- Check if the network connection is normal
- Contact Mojang Support

## Network Issues

### Slow Download Speed

**Problem**: Slow download speed when downloading game versions or mods.

**Solutions**:
- Check if the network connection is normal
- Close other network-intensive programs
- Try changing the network environment
- Try changing the download source in settings
- Try using a VPN/accelerator

## Other Issues

### Save Files Lost

**Problem**: Game save files are lost.

**Solutions**:
- Try searching in the .Minecraft/your-version/(ignore if version isolation is disabled)/saves/ folder

### Mods Not Working

**Problem**: Installed mods are not working properly.

**Solutions**:
- Check if the mod is compatible with the game version
- Check if the mod conflicts with other mods
- Try updating the mod in version management
- Check if the mod dependencies are installed
- Check the mod's documentation
