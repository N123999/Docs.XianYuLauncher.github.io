# Launch Minecraft

## Select Game Version

1. Select the version you want to launch from the version list on the launcher's launch page
2. Click the version name to select it

## Launch the Game

1. Make sure you have added a Minecraft character
2. Make sure you have selected the correct game version
3. Click the "Launch Game" button
4. Wait for the game to load
5. Enter the Minecraft main interface

## First Time Entering the Game

1. Select "Singleplayer" or "Multiplayer"
2. If you want to play singleplayer, you can:
   - Create a new world
3. If you want to play multiplayer, you can:
   - Add a server

## Common Issues

### Game Failed to Launch

- Check if you have installed the correct Java version (Java 8 for lower versions, Java 21+ for higher versions recommended)
- Check if the game memory allocation is reasonable
- Check if you have installed the required drivers
- View the launcher logs for detailed error information, then join the feedback group: 1002593659 and contact @咸鱼

### Game Crashes

- Check if you have installed incompatible mods
- Check if the game version matches the loader version
- View crash reports for detailed information
