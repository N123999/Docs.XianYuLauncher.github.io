# Multiplayer

## Join Servers

### Direct Connection

1. Launch Minecraft and enter the main interface
2. Click "Multiplayer"
3. Click "Direct Connect"
4. Enter the server address
5. Click the "Connect" button
6. Wait to connect to the server

### Add Server to List

1. Launch Minecraft and enter the main interface
2. Click "Multiplayer"
3. Click "Add Server"
4. Enter the server name
5. Enter the server address
6. Click the "Done" button
7. The server will be added to the server list
8. Click the server name to connect

## Terracotta Multiplayer Service

::: info
The multiplayer feature of XianYuLauncher is powered by **Terracotta Multiplayer** service.
:::

XianYuLauncher has built-in multiplayer service, allowing you to easily play with friends:

1. On the launcher main interface, click the "Multiplayer" icon in the sidebar
2. You can:
   - Create a room
   - Join friends' rooms
   - View room members
3. On the launch page, click the "Launch Game" button
4. After entering the game, click "Multiplayer", a prominent room area will appear, click "Join Room" to join the room
