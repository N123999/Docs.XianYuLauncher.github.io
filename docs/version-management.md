# 游戏版本管理

## 查看版本列表

1. 在启动器版本列表页，会显示所有已安装的版本

## 编辑版本

1. 在版本列表中选择一个版本，打开版本管理页
2. 在版本管理页，可管理以下版本属性：
   - 游戏内存分配
   - Java 路径
   - 游戏窗口设置
   - Mod、资源包、光影等

## 删除版本

1. 在版本列表中选择一个版本然后右键
2. 在版本管理页，点击 "删除" 按钮
3. 在确认对话框中，点击 "确定" 按钮
4. 该版本会从列表中删除
5. 注意：删除版本会删除所有相关文件，包括存档和模组

## 导入/导出整合包

### 导出整合包

1. 在版本列表中选择一个版本
2. 右键点击版本，选择 "导出整合包"
3. 选择要导出整合包的内容
4. 点击 "导出" 按钮
5. 等待导出完成

### 导入整合包

1. 在版本列表区域，点击 "导入整合包" 按钮
2. 选择要导入的整合包文件
3. 点击 "打开" 按钮
4. 等待导入完成
5. 导入的整合包会出现在版本列表中
